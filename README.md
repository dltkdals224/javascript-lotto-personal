## ⌨️ 실행 방법

Clone the project

```bash
  git clone https://github.com/dltkdals224/javascript-lotto-personal.git
```

Go to the project directory

```bash
  cd javascript-lotto-personal
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start-be
```

```bash
  npm run start-fe
```
